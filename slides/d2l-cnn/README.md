# CNN: From Fully Connected Layers to Convolution

Dive into Deep Learning 輪読ゼミで作成した、CNNの導入に関する発表資料です。

## Overview

画像認識で全結合層ではなく畳み込み層を使う理由を、平行移動等価性、局所性、重み共有という観点から整理しました。

また、相互相関演算、エッジ検出、パディング、ストライドなど、CNNの基本的な計算を数式と図で説明しています。

## Keywords

- Inductive Bias

- Locality

- Cross-Correlation
