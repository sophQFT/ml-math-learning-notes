# Kubokawa Mathematical Statistics Notes

久保川『現代数理統計学の基礎』を学習する中で、定義・定理・証明の行間を自分の理解のために補った学習ノートです。

## Topics

- 統計量と標本分布
- 推定量と不偏性
- 標本平均・標本分散・不偏分散
- 正規母集団における標本平均と不偏分散の独立性
- ヘルマート行列を用いた独立性の証明
- t分布
- 確率収束
- 大数の弱法則
- 分布収束
- 中心極限定理
- 連続写像定理
- スラツキーの定理
- 順序統計量

## Structure

- `main.tex`: 全体をコンパイルするメインファイル
- `preamble.tex`: パッケージ・定理環境などの設定
- `section1.tex`: 統計量・標本分布と推定量
- `section2.tex`: 標本平均・標本分散の独立性と分布
- `section3.tex`: 確率変数列の収束と漸近理論
- `pdf/`: コンパイル済みPDF

## Motivation

機械学習や不確実性モデリングを、確率論・数理統計の基礎から理解するために作成しました。特に、正規母集団における標本平均と不偏分散の独立性、中心極限定理、確率収束・分布収束の関係などを、証明の行間を追いながら整理しています。

## Notes

この資料は個人の学習ノートです。内容には誤りを含む可能性があります。公開にあたっては、教科書本文の転載ではなく、自分の理解に基づく行間補足・導出整理として作成しています。
